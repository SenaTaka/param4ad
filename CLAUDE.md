# CLAUDE.md — param4ad

ラズパイ搭載 LiDAR 自動運転ロボット（Follow the Gap）を Vercel Web UI からリアルタイムにパラメータ制御するシステム。

- **Web UI**: https://param4ad.vercel.app
- **ロボット側**: `param1.py`（ラズパイで実行）
- **通信**: Next.js App Router + Upstash Redis
